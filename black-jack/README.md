# Blackjack

A simple Blackjack game I built while following along with the Scrimba Full Stack course.

## How it works

- Click **START GAME** to begin
- You get dealt two cards
- Your sum is calculated automatically
- If you hit 21 — you win. If you go over — you're out.

## How to run

1. Open `index.html` in your browser
2. Click the button and play

## What I learned

- Variables and conditionals
- Functions
- Basic DOM manipulation
- Working with the DOM to update the UI
